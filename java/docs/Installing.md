# Installing

=== "Gradle"
    ```kotlin
    implementation("com.grafana:grafana-foundation-sdk:10.1.0-1745704785")
    ```
=== "Maven"
    ```xml
    <dependency>
        <groupId>com.grafana</groupId>
        <artifactId>grafana-foundation-sdk</artifactId>
        <version>10.1.0-1745704785</version>
    </dependency>
    ```
