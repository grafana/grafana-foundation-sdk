# Installing

=== "Gradle"
    ```kotlin
    implementation("com.grafana:grafana-foundation-sdk:11.4.0-1744895652")
    ```
=== "Maven"
    ```xml
    <dependency>
        <groupId>com.grafana</groupId>
        <artifactId>grafana-foundation-sdk</artifactId>
        <version>11.4.0-1744895652</version>
    </dependency>
    ```
