# Installing

=== "Gradle"
    ```kotlin
    implementation("com.grafana:grafana-foundation-sdk:11.0.0-1738846382")
    ```
=== "Maven"
    ```xml
    <dependency>
        <groupId>com.grafana</groupId>
        <artifactId>grafana-foundation-sdk</artifactId>
        <version>11.0.0-1738846382</version>
    </dependency>
    ```
