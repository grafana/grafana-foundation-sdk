# <span class="badge object-type-enum"></span> ThresholdsMode

Thresholds can either be `absolute` (specific number) or `percentage` (relative to min or max, it will be values between 0 and 1).

## Definition

```typescript

	Absolute = "absolute",
	Percentage = "percentage",
}
```
