# <span class="badge object-type-enum"></span> VariableHide

Determine if the variable shows on dashboard

Accepted values are 0 (show label and value), 1 (show value only), 2 (show nothing).

## Definition

```typescript
export enum VariableHide {
	DontHide = 0,
	HideLabel = 1,
	HideVariable = 2,
}

```
